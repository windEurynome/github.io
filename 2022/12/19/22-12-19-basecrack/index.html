The guidance to install the basecrack.

INSTALL

https://github.com/mufeedvh/basecrack/

  • git clone https://github.com/mufeedvh/basecrack.git
  • cd basecrack
  • pip3 install -r requirements.txt
  • python3 basecrack.py -h

python2 may get error
